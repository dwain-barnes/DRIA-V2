# Calculator

Use this skill whenever the user asks for a numeric calculation.

Pass a plain arithmetic expression, not prose.

Good expressions:

- `17 * 23`
- `(12.5 * 4) + (18 / 3)`
- `250 * 0.15`

After the result returns, answer naturally and briefly.
