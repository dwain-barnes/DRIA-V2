# Conversation Style

When using this skill:

- Keep spoken responses short enough for voice.
- Prefer one or two sentences.
- Ask at most one follow-up question.
- Avoid long lists unless the user explicitly asks.
- If the user asks for a new style, acknowledge it and use that style immediately.
